# プロジェクト名
ChanPla

# 概要
使用するには、ユーザー登録をします。<br>
登録したユーザーでログインしこのアプリを使用開始します。<br>
Youtubeのチャンネル、動画を登録できます。<br>
お気に入り共有を有効にすれば、他のユーザーとお気に入り共有できます。<br>
お気に入り共有した他のユーザーとチャットができます。<br>
